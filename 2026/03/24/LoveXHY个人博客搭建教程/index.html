LoveXHY个人博客搭建教程

本文适用于零基础新手，详细讲解基于Hexo框架搭建个人博客的完整流程，包含环境配置、项目初始化、本地预览、部署上线全步骤。

一、前置准备

1. 安装必备软件

  • Node.js：Hexo基于Node.js运行，推荐安装LTS版本（下载地址），安装后执行node -v验证是否安装成功；
  • Git：用于代码管理和部署，下载地址，安装后执行git --version验证；
  • 代码编辑器（可选）：推荐VS Code，方便编辑博客文章和配置文件。

2. 注册代码托管平台账号

推荐GitHub（国际）或Gitee（国内），用于存放博客源码和部署静态页面（后续以GitHub为例）。

二、核心搭建步骤

1. 安装Hexo-CLI（Hexo命令行工具）

打开终端/命令提示符，执行以下命令全局安装：
```bash
npm install -g hexo-cli
